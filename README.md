## Features
* Power Options
    * Respring
    * Soft Reboot (userspace reboot)
    * Full Reboot
* Location Services
    * Set Location Services
    * Simulation Location
* Flashlight Control
    * Enable Amber
    * Enable White
    * Enable Both
* Display Options
    * Set Auto-Brightness
    * Set Auto-Lock Duration
* iCloud Backup Options
    * Start iCloud Backup
    * End iCloud Backup
    * Get Status of iCloud Backup
* Set Ringer On/Off
